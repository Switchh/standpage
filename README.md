# StandPage

StandPage is a simple reusable PHP holding page for parked domains, standby websites and upcoming projects.

It provides a clean responsive landing page with a friendly illustration, automatic HTTPS detection and a lightweight file-based visit counter. No database, no admin panel, no setup complexity.

## Features

- Simple PHP single-page setup
- Responsive waiting page layout
- Lightweight visit counter using a text file
- HTTPS detection with proxy/CDN support
- No database required
- Easy to customize
- Suitable for parked domains, client placeholders and temporary project pages

## Installation

Upload the following files to the root of your website:

index.php
stand-fox-vert.webp
counter.txt
